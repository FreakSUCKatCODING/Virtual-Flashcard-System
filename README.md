# Virtual-Flashcard-System
📂 Virtual Flashcard System

A simple, responsive Virtual Flashcard System built for students to organize and revise study material efficiently. All data is stored locally on your computer, ensuring privacy and fast access.

🚀 Features
📁 Create folders and nested folders
🧠 Add and manage flashcards easily
💾 Local storage (no internet/database required)
📱 Fully responsive design
⚡ Fast and lightweight
📌 Use Case

This project is designed for students who want to:

Organize subjects like Maths, Science, etc.
Create structured notes using folders
Revise concepts quickly using flashcards
Study offline without losing data
🛠️ How to Use
Open the Application
Launch the project in your browser.
Create a Folder
Enter a folder name in the input field
Click + Folder
Example: science
Navigate Folders
Click on a folder (like maths or science) to open it
Use ← Back to go to the previous folder
Use 🏠 Home to return to the main directory
Create Nested Folders
Inside any folder, repeat the same process to create subfolders
Add Flashcards
Click + Flashcard
Enter your question/answer (depending on your implementation)
Delete Items
Click the ❌ button next to folders/flashcards to remove them
💡 Example Workflow
Create a folder: Science
Inside it, create: Physics, Chemistry
Add flashcards like:
What is gravity?
Define atom
⚙️ Tech Stack
HTML
CSS (Responsive UI)
JavaScript (Local Storage)
🔒 Data Storage
All data is stored in your browser using localStorage
No external server or database is used
Your data stays on your device
